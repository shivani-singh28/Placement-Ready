Capgemini Exceller Behavioral Competency


This section will judge your situation analyzing and decision-making skills. Do not get confused by the name of this section, this is just another name for the Psychometric Test. There will be situation-based questions in this test, which you have to agree or disagree with.


One of the most essential things to remember when answering these questions is to remember the answer you choose for each question. Because there is a risk that the same question will be given in a different way later in the section, and if your answers do not match the two questions, you may fail this exam.

Capgemini Exceller Game based Aptitude Test


This is all together a new section in the selection process of Capgemini Exceller for 2023 passouts. In this section there are a total of 24 games, out of which the system will select 4 games for you. Which will test your multitasking, memorizing, analyzing and logical thinking abilities. Below starting each game there will be a demo for that game, which you can watch as many times as you want to. For each game there will be a time limit of 5-6 mins, in which you have to reach the maximum level which you can.

Some of the games from which the system will select are-:


  • Deductive Logical Thinking(Geo-Sudo)
  • Inductive-logical Thinking
  • Grid Challenge
  • Motion Challenge
  • Switch Challenge
  • Digit Challenge

Capgemini Exceller Game Based Analysis:
